Utilise une instance de WordPress en local

Dans le dossier où WordPress installe ses extensions, ajoute un nouveau dossier partenariat-campus

Où WordPress installe ses extensions ? WordPress installe des plugins dans wp-content/plugins/

Dans le dossier partenariat-campus crée un nouveau fichier, appele-le partenariat-campus.php

Il n'est pas obligatoire que le fichier soit nommé comme le dossier, mais c'est une convention très souvent respectée.

Dans cette étape tu vas faire ce qu'on l'appele déclarer le plugin.

Tu va éditer le fichier partenariat-campus.php. Il s'agit de renseigner le nom de ton plugin ainsi que d'autre informations le concernant (auteur, version, descriptions, etc.) Tu peux trouver plus d'information par exemple dans un tuto sur OpenClassrooms

La version minimum de la déclaration d'un plugin est comme ceci : 
<?php
/*
Plugin Name: Partenariat Campus
*/

Suite à l'étape précédente tu devrais appercevoir ton plugin dans ton Tableau de Bord > Extensions Installées. Vérifie que ton plugin y est bien présent. Visite ton Dashboard > Extensions Installées

Rends ton plugin fonctionnel.

Ici, tu devrais enregistrer un nouveau shortcode et définir ce qu'il affiche.

exemple

Voici l'exemple d'un shortcode qui permet d'afficher un paragraphe Hello World! I ♥ you! avec un shortcode [helloworld].

function HelloWorldShortcode() {
    return '<p>Hello World! I &hearts; you!</p>';
}
add_shortcode('helloworld', 'HelloWorldShortcode');

Vérifie que ton plugin fonctionne. Ajoute le shortcode [partenariat-campus] dans plusieurs articles.

Si cela fonctionne, bravo ! Tu viens de créer un extension WordPress ! Si tu as envie de continuer, allons-y !

Ajoute une mise en page et un logo du campus.

Dans le dossier de ton extension crée le ficher style.css. Ensuite ajoute un dossier images. Mets le logo dans le dossier images.

Charge le fichier style.css dans la partie head de la page

function partenariat_campus_style() {
  wp_enqueue_style( 'partenariat-campus-stylesheet', plugins_url( 'partenariat-campus/style.css' ) );
}
add_action( 'wp_enqueue_scripts', 'partenariat_campus_style' );

plugins_url( 'partenariat-campus/style.css' )
correspond à
http(s)://mon-adresse-du-site/wp-content/plugins/partenariat-campus/style.css
