Utilisez une instance de WordPress en local (par exemple, migrez trello-shop en local)

Dans le dossier où WordPress installez ses extensions, ajoutez un nouveau dossier hide-flat-rate-woocommerce-addon

Où WordPress installe ses extensions ? Dans wp-content/plugins/

Un peu de lecture préparatoire ? Par exemple ceci ?

Dans le dossier hide-flat-rate-woocommerce-addon créez un nouveau fichier, appelez-le hide-flat-rate-woocommerce-addon.php

C'est une convention de nommer le fichier principal du plugin comme le dossier.

Déclaration du plugin

Il s'agit de renseigner le nom de ton plugin ainsi que d'autre informations le concernant (auteur, version, descriptions, etc.)

Suite à l'étape précédente vous appercevez votre plugin dans ton Tableau de Bord > Extensions Installées. Vérifiez que le plugin y est bien présent.

Plugin fonctionnel.

Vous pouvez trouver le snippet PHP qui permet de masquer toutes les autres méthodes et d'afficher uniquement la livraison gratuite dans la documentation officielle de WooCommerce.

Chercher par exemple hide flat rate when free shipping woocommerce

Vérifiez que le plugin fonctionne correctement.
